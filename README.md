# 💧 Smart RO Filtration Water Plant

## 📌 Project Overview
This project is a smart water delivery system designed to solve local water supply challenges. It allows users to order 20-liter water cans based on their live location, ensuring service availability within a defined radius.

---

## 🚀 Features
- 📍 Live location-based ordering
- 🕒 Service available from 6 AM to 8 PM
- 📏 Distance calculation using geolocation
- 🚫 Restricts service outside delivery area (5 km radius)
- 📱 Simple and responsive user interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Geolocation API)

---

## ⚙️ How It Works
1. User enters number of water tins
2. Clicks on "Send Live Location & Order"
3. System fetches user's live location
4. Calculates distance from plant location
5. If within 5 km → Order accepted
6. If outside → Service denied

---

## 📍 Use Case
This system is useful for:
- Local water supply businesses
- Rural and semi-urban water distribution
- Smart delivery services
